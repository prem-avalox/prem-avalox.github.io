    Acerca de Easy Access

    Pensada Windows 11, Easy Access es una ligera herramienta diseñada para puntos de venta de tiendas pequeñas y medianas, que permite administrar productos y gestionar las credenciales de los trabajadores de manera rápida y segura. La aplicación utiliza scanners de código de barras, evitando tener que ingresar manualmente una credencial empresarial completa. Además, permite instalar scanners en distintas secciones de la tienda para que los clientes puedan consultar precio de los productos de forma inmediata, y facilita agregar o eliminar trabajadores según las necesidades del negocio.

  • Sistema de Gestión de Exportación de Flores (SGEF) (JavaFX, PostgreSQL)

    Desarrollo de interfaz con SceneBuilder y MaterialFX Implementación de módulos de clientes, ventas y pedidos Validación de formularios.

    Acerca de SGEF

    Diseñado para optimizar la gestión de exportación de flores, el Sistema de Gestión de Exportación de Flores (SGEF) es un sistema ligero y eficiente desarrollado para INDEXBLOOMS S.A.S.. Permite administrar clientes, pedidos, ventas y stock de manera centralizada, garantizando un flujo de trabajo ágil y seguro. El sistema facilita la creación de pedidos detallados por variedad, largo y tipo de empaque, calcula automáticamente subtotales, IVA y totales, y permite generar reportes de inventario y ventas al instante. Además, SGEF cuenta con módulos de gestión de usuarios, asignación de roles y control de credenciales, asegurando que cada trabajador acceda solo a las funciones necesarias según su puesto. Su interfaz intuitiva y personalizable permite operar de manera rápida y confiable, optimizando la eficiencia en cada etapa del proceso de exportación de flores.
